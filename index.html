This training is designed to equip you with the knowledge and skills to navigate and utilise the K1 CRM system (known in the market as QContact). By mastering the system, you'll be able to maintain accurate records, and manage communication efficiently across all channels.

About QContact

QContact is a comprehensive communication platform that unifies calls, emails, live chat, WhatsApp, social media, and more into a single interface. It ensures that all customer interactions are recorded systematically and managed efficiently, providing a seamless experience for both agents and customers.

Core Purposes of the K1 System

The K1 CRM is a key tool that supports three main functions:

  • Communication: Organising all client and internal interactions in one place.
  • Record-Keeping: Maintaining accurate, up-to-date information for quick access and reference.
  • Managing Queries: Tracking, prioritising, and resolving client queries efficiently.

Search Function

  1. Open the search bar:

    Click the Search button (Magnifying glass icon).

  2. Enter search information:

    Type any known details of the contact.

  3. Searchable fields include:
    • Name
    • Surname
    • ID Number
    • Membership Number
    • Telephone Number
    • Email Address
    • Payroll Number
    • Reference Number

Contacts
